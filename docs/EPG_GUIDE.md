# 📺 EPG (Electronic Program Guide) - Полное руководство

## 🎯 Что такое EPG?

EPG (Electronic Program Guide) - это электронная телепрограмма, которая показывает:
- Что идёт **сейчас** на канале
- Что будет **дальше** (программа на 7+ дней)
- Описание передач, время начала/окончания
- Постеры, категории, рейтинги

---

## 🔄 Как работает EPG

```
┌─────────────┐      ┌──────────────┐      ┌─────────────┐
│   iptv-org  │─────▶│  EPG Grabber │─────▶│  guide.xml  │
│ guides.json │      │   (parser)   │      │   (XMLTV)   │
└─────────────┘      └──────────────┘      └─────────────┘
                             │                      │
                             │                      ▼
                             │              ┌─────────────┐
                             │              │ IPTV Player │
                             │              │ (VLC/Kodi)  │
                             │              └─────────────┘
                             ▼
                     ┌──────────────┐
                     │ channels.xml │
                     │   (config)   │
                     └──────────────┘
```

### Шаги:

1. **guides.json** содержит карту "канал → EPG источник"
2. **EPG Grabber** скачивает программу с официальных сайтов
3. **channels.xml** - конфигурация (какие каналы парсить)
4. **guide.xml** (XMLTV) - результат парсинга
5. **IPTV Player** читает guide.xml и показывает программу
